## 📅 프로젝트 기간

<br>

## ✨ 프로젝트 핵심 기능

#### 💬 주문 기능
- 상품을 장바구니 추가, 삭제할 수 있습니다
- 상품 주문, 조회, 취소가 가능합니다
- 관리자는 상품 주문 수정, 삭제가 가능합니다

#### 🔎 상품 기능
- 사용자는 상품을 카테고리별, 상세페이지를 통해 제품을 확인할 수 있습니다
- 관리자는 관리자 계정을 통해 상품과 카테고리를 추가, 수정, 삭제가 가능합니다

#### 🙌 유저 기능
- 회원가입, 로그인, 회원정보 수정이 가능합니다
